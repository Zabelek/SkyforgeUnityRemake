# Changelog

## v0.0.2a
- Reworked the debug window into a proper game menu, looking similar to the one from the original game
- Moved the settings management entirely into the separate game menu scene. The gameplay scene, however, still needs to have SettingsManager, to apply the current local settings into the scene
- Removed the debug perk check boxes
- Moved the perks into the SkyforgeLoader so that they are consistent across the player profile
- LockablePerk.Unlocked variable is removed. Now all perks added to _perks list are treated as unlocked. Locking them means removing them from the list. They can still be enabled or disabled normally.
- Modified perks can be now saved in the player profile so it's not needed to unlock them every time the player enters the game
- Changed the way perks are identified into a string variable ID - this is also the identifier in the save game file
- Added KillOffCombat() method to properly kill characters without need to damage them first
- Reduced the amount of iterations called OnPerkChange in the PlayerBehaviour class
- Perk effects now can be modified in Addition mode, meaning, there's no need to reset the character stats every time perks are changed
- Massive changes in the ChoosablePerkSet class. It's not applicable to the player now, because the player uses SkyforgeLoader for that purpose
- Reworked perk adding/removing logic from HeroBehaviour
- Added PerkRegistry Addressable, it will soon replace Globals.RegisteredPerks entirely
- Fixed some typos in tooltips to the custom classes
- Reworked the SettingsWindow and the DebugWindow to properly work with new menu. Removed them from the Divine Observatory scene
- Enhanced the Ascension Atlas view background visuals
- Added Eidoses and Credits visuals
- Added Eidoses and Credits counters in the Game Menu
- Added Eidos vending machine mesh to the Divine Observatory scene
- Added interact animation for the regular human female animations and for the berserker female animations (the bug from the original game when the character on interaction with weapon drawn would do weird things with the weapon, is now a history, as the classes will have their own interact animations)
- Added Interactable objects interface
- Created interactable function to the eidos vending machine
- Added GameplayResources to the PlayerProfile
- Added profile name and prestige displayed in the game menu
- Added a simple scatter script, helping with random distribution of objects into the scene
- Fixed a bug when the freshly created profile would not sync with the gameplay scene
- Fixed a bug with the damage display, preventing it to fade out properly
- Added resource state change side notifications

- Implemented some of the code improvements and bug fixes from the AleksSmash2019’s fork. Thank you again for your amazing work, AleksSmash2019!
- Implemented vsync, window mode and target frame rate settings by AleksSmash2019
- Introduced IDs to some Scriptable Objects, as they duplicate in some cases and SO = SO doesn’t always work. All Scriptable Object will have string IDs soon.
- Added Atlas Node class
- Added small notification animation of changing resources on the top of the resource counters in the bottom of the game menu
- Added syncing node unlock states with the player acquired perks
- Added a tooltip window class
- Added nodes instantiate tooltips on mouse hover
- Tooltip follows the mouse cursor and repositions itself depending on screen edge distance. 
- Added visual lines connecting parent-child Atlas nodes.
- Added parent-child unlock restrictions to the nodes
- Added visual unlock animation to the nodes
- Added sound effects to the Ascension Atlas view
- Added sound effects to some buttons in GUI
- Added camera movement bounds in the Atlas view
- Added separate atlases for the character and the Berserker class, switchable by the bottom-left buttons.
- Added stat types: Max Damage, Vampirism, Defense, Stability
- Added full Berserker and the character atlases with their nodes
- Increased the difficulty level of the Divine Observatory Test dungeon to better align with node improvements
- Added Abilities panel prototype, allowing to change the Terra perks (replacement for a debug window)
- Fixed a bug causing the enemies to incorrectly scale their stats with a difficulty level
- Changed the Argus voice at the end
- Minor code cleanup
